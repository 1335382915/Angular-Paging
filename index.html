Angular Paging

A directive to aid in paging large datasets while requiring the bare minimum of actual paging information. By using AJAX, hybrid server/client-side pagination can be used, in which Javascript is used to request the subsequent page which is loaded and inserted into the Document Object Model via AJAX.
